<p><p>Container Gantry Cranes market types include Single Main Girder and Double Main Girder cranes. Single Main Girder cranes have one main beam that supports the hoist, while Double Main Girder cranes have two main beams for added stability and capacity. Single Main Girder cranes are more commonly used for lighter loads and smaller spaces, while Double Main Girder cranes are preferred for heavier loads and larger containers. Both types offer efficient and reliable solutions for container handling in ports and industrial settings.</p></p>
